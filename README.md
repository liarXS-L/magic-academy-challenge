# 🧙‍♂️ 魔法学院大挑战

一款基于 Taro 4.x + React 18 + TypeScript 构建的跨平台消除匹配小游戏。

## ✨ 功能特性

- 🎮 **消除玩法**：9×9 棋盘，点击交换相邻元素完成消除
- 🔗 **连锁消除**：消除后自动触发连锁反应（Combo）
- 🎯 **限时挑战**：每关 50~60 秒，追求更高评价
- 📚 **课程系统**：4 门魔法课程，每门 4 个关卡
- 🧙 **角色系统**：4 个魔法师角色，各有专属技能加成
- 🎨 **魔法视觉**：暗黑奇幻风格，霓虹紫渐变，发光特效
- 💥 **特殊道具**：四消生成条纹道具、十字消生成炸弹、五消生成彩虹道具

```

├── app.config.ts              # 全局路由配置

├── app.scss                   # 全局样式
├── styles/                    # 样式库

├── types/

├── data/
│   └── gameData.ts            # 课程/角色/关卡 Mock 数据
├── utils/
│   └── gameLogic.ts           # 游戏核心逻辑（棋盘/消除/计分）
├── components/                # 组件
│   ├── GameBoard/             # 游戏主棋盘
│   ├── GameElement/           # 单个魔法元素

│   ├── ScoreDisplay/          # 分数展示
```

## 🎮 游戏玩法

1. **选择课程**：在首页选择一门魔法课程
2. **选择角色**：选择一个魔法师角色
3. **选择关卡**：进入关卡选择页，选择要挑战的关卡
4. **开始游戏**：点击"开始挑战"进入游戏
5. **消除元素**：点击选中元素 → 点击相邻元素交换位置
6. **完成目标**：在限定时间内达到目标分数
7. **查看结果**：游戏结束后展示评价等级（S/A/B/C/D）

## 💥 特殊消除效果

| 消除类型 | 道具效果 |
|----------|----------|
| 四消 | 生成条纹道具，消除整行或整列 |
| 十字消 | 生成炸弹道具，消除 3×3 区域 |
| 五消 | 生成彩虹道具，消除所有同类型元素 |
