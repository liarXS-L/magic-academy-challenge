# 🎁 魔法水晶奖励系统 - 设计文档

## 一、概述

本设计文档描述魔法学院大挑战游戏的奖励系统，核心是「魔法水晶」货币系统，配合游戏内道具，增强玩家的游戏能力和体验。

## 二、奖励类型

| 奖励 | 图标 | 类型 | 获取方式 | 使用方式 |
|------|------|------|----------|----------|
| 🔮 魔法水晶 | 💎 | 货币 | 通关奖励、成就奖励 | 商店兑换道具 |
| ⚡ 能量药剂 | 🧪 | 道具 | 游戏中随机掉落、商店购买 | 主动使用，立即+500分 |
| 🕐 时间沙漏 | ⏳ | 道具 | 游戏中随机掉落、商店购买 | 主动使用，增加30秒时间 |
| 📈 分数卷轴 | 📜 | 道具 | 游戏中随机掉落、商店购买 | 主动使用，10秒内得分2倍 |

## 三、获取方式

### 3.1 通关奖励（根据评价）

| 评价 | 魔法水晶数量 |
|------|-------------|
| S | 100 |
| A | 60 |
| B | 40 |
| C | 20 |
| D | 10 |

### 3.2 游戏中随机掉落

- 消除3个及以上元素时有概率掉落道具
- 掉落概率根据消除数量动态计算：消除越多概率越高
- 掉落类型随机（能量药剂/时间沙漏/分数卷轴）

### 3.3 成就奖励

| 成就ID | 成就名称 | 达成条件 | 奖励 |
|--------|----------|----------|------|
| first_win | 首次胜利 | 完成第一次通关 | 100魔法水晶 |
| combo_master | 连击大师 | 达成10连击 | 50魔法水晶 |
| total_matches_1000 | 消除大师 | 累计消除1000个元素 | 200魔法水晶 |
| s_grade_10 | S级专家 | 累计获得10次S级评价 | 500魔法水晶 |
| complete_all_courses | 全能法师 | 完成所有课程的所有关卡 | 1000魔法水晶 |

## 四、使用方式

### 4.1 商店系统

在首页添加「商店」入口，可用魔法水晶兑换道具：

| 道具 | 价格（魔法水晶） | 效果 |
|------|-----------------|------|
| ⚡ 能量药剂 | 50 | 立即获得500分 |
| 🕐 时间沙漏 | 30 | 增加30秒游戏时间 |
| 📈 分数卷轴 | 80 | 10秒内得分翻倍 |

### 4.2 游戏中主动使用

游戏页面底部添加道具栏，点击道具即可使用：
- 道具显示数量（如 `🧪 x3`）
- 使用后数量减少
- 使用后有30秒冷却时间（避免滥用）

## 五、数据结构

### 5.1 类型定义

```typescript
export type ItemType = 'energy_potion' | 'time_hourglass' | 'score_scroll';

export interface InventoryItem {
  type: ItemType;
  count: number;
}

export interface Achievement {
  id: string;
  name: string;
  description: string;
  condition: string;
  reward: number;
  isCompleted: boolean;
  progress: number;
  target: number;
}

export interface PlayerData {
  crystals: number;
  inventory: InventoryItem[];
  achievements: Achievement[];
  totalMatches: number;
  totalSGrades: number;
}
```

### 5.2 数据存储

使用 `Taro.setStorageSync` / `localStorage` 存储 `playerData` 对象。

## 七、交互流程

### 7.1 通关奖励发放

游戏结束 → 计算奖励 → 更新 `playerData` → 结果页显示获得的奖励和成就

### 7.2 游戏中道具掉落

消除完成 → 计算掉落概率 → 生成道具 → 更新 `playerData` → 显示掉落提示

### 7.3 商店购买

进入商店 → 选择商品 → 检查水晶数量 → 扣除水晶 → 更新 `playerData` → 显示购买成功提示

### 7.4 游戏中使用道具

点击道具 → 检查数量和冷却 → 应用效果（加分/加时/倍率）→ 更新数量 → 开始冷却计时

## 八、成就系统

### 8.1 成就追踪

- `totalMatches`：每次消除增加对应分数
- `totalSGrades`：获得S级评价时增加1
- `completeAllCourses`：检查所有课程进度
- `first_win`：首次通关时解锁
- `combo_master`：记录最大连击数

### 8.2 成就解锁

达成条件时自动解锁，奖励自动发放，在结算页面显示解锁提示。

## 十、特殊元素组合效果

### 10.1 组合规则

当玩家将两个特殊元素互换时，触发组合效果，无需通过匹配消除。

### 10.2 组合效果列表

| 组合 | 效果名称 | 效果描述 |
|------|----------|----------|
| 横条纹 + 竖条纹 | 🌈 十字风暴 | 消除交叉的整行和整列（共17个格子） |
| 炸弹 + 条纹 | 💥 冲击波 | 消除 5×5 区域，呈十字冲击波扩散 |
| 炸弹 + 炸弹 | 🔥 超新星爆发 | 消除 5×5 区域，并在四周生成小炸弹 |
| 彩虹 + 条纹 | ✨ 彩虹光束 | 消除整行/整列，并将该行/列所有元素变成随机特殊道具 |
| 彩虹 + 炸弹 | 🌀 次元裂隙 | 消除 3×3 区域，并消除棋盘上最常见的一种类型元素 |
| 彩虹 + 彩虹 | ⚡ 混沌风暴 | 消除棋盘上所有元素，重新生成全新棋盘 |

### 10.3 触发流程

1. 检测两个互换元素是否都有特殊属性
2. 调用 `triggerSpecialCombo` 判断组合类型
3. 标记受影响格子为 `isMatched`
4. 处理变换（如超新星生成小炸弹）
5. 执行消除和连锁反应
